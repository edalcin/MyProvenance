# Introduction

MyProvenance records the provenance of biodiversity data: where it came from, who produced it, and what transformations and analyses it went through before becoming the result you cite in a paper or report.

![Provenance diagram generated from this manual's full example](/manual/en/00-diagrama-exemplo.png)

## Why record provenance

- **Reproducibility.** An Analysis with no record of which Entity it used and which Process it applied cannot be redone or checked.
- **Trust.** Whoever reads a result needs to know who produced it (Agent), with what (Tool or Software) and when.
- **Traceability.** A suspicious data point should be traceable back to the raw spreadsheet and the collection activity that originated it.
- **Attribution.** Every Entity and Activity is associated with an Agent â€” person, institution or software â€” preserving credit.
- **Journal and funder requirements.** Data availability and provenance statements are increasingly required when submitting papers and project reports.

## What being W3C PROV-compatible means

[W3C PROV](https://www.w3.org/TR/prov-overview/) is the standard model for representing provenance on the Web. MyProvenance does not implement PROV-O literally in the database, but every domain concept has a direct equivalent in the model â€” which lets you export or map the data to PROV whenever needed, without redesigning anything.

| MyProvenance                                    | W3C PROV                                                                  |
| ----------------------------------------------- | ------------------------------------------------------------------------- |
| Entity                                          | `prov:Entity`                                                             |
| Activity (Creation / Transformation / Analysis) | `prov:Activity`                                                           |
| Agent â€” Person / Institution / Software         | `prov:Agent` (`prov:Person` / `prov:Organization` / `prov:SoftwareAgent`) |
| Entity generated by an Activity                 | `prov:wasGeneratedBy`                                                     |
| Activity uses an Entity                         | `prov:used`                                                               |
| "Derivation" origin relationship                | `prov:wasDerivedFrom`                                                     |
| "Revision" origin relationship                  | `prov:wasRevisionOf`                                                      |
| Activity carried out by an Agent                | `prov:wasAssociatedWith`                                                  |
| Entity attributed to an Agent                   | `prov:wasAttributedTo`                                                    |
| Provenance Record (graph container)             | `prov:Bundle`                                                             |
