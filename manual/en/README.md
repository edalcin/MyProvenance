# User manual

Complete guide to MyProvenance: from creating a Provenance Record to the final export, with an end-to-end example using a real biodiversity dataset. Each page covers one feature and connects the action in the interface to the matching term in [W3C PROV](https://www.w3.org/TR/prov-overview/).

1. [Introduction](/como-usar/00-introducao) — why record provenance and how MyProvenance maps to W3C PROV.
2. [Create a record](/como-usar/01-criar-registro) — starts a new Provenance Record as a draft.
3. [Add an Activity](/como-usar/02-adicionar-atividade) — Creation, Transformation and Analysis: how the lineage grows.
4. [Edit Record and Activity](/como-usar/03-editar) — what can be changed, and until when.
5. [Finalize](/como-usar/04-finalizar) — make the history immutable.
6. [Export JSON](/como-usar/05-exportar-json) — full portable copy of the data.
7. [Export .md report](/como-usar/06-exportar-relatorio-md) — human-readable report with a diagram.
8. [Share](/como-usar/07-compartilhar) — public read-only link and co-editing with other accounts.
9. [Delete a record](/como-usar/08-excluir-registro) — permanent, cascading removal.
10. [Full example](/como-usar/09-exemplo-completo) — raw spreadsheet → Darwin Core → analyses, start to finish.
